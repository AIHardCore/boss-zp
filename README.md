# BOSS直聘 + 企查查 数据采集工具

## 项目简介

本项目实现BOSS直聘和企查查的数据采集与整合，可用于招聘数据分析、企业信息调查等场景。

## 功能特点

- ✅ BOSS直聘多关键词批量采集
- ✅ 广州兼职职位筛选
- ✅ 自动获取职位详细信息
- ✅ 获取发布人信息（名称、职称、活跃状态）
- ✅ **点击进入公司主页获取详细企业信息**
- ✅ 企查查公司信息采集
- ✅ 一般纳税人资质筛选
- ✅ 数据自动整合

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置

编辑 `config.py`：

```python
# 搜索关键词
SEARCH_QUERIES = ['软件开发', '云服务', '短视频运营', '会展活动', '咨询培训']

# 城市代码（广州）
CITY_CODE = '101280100'

# 职位类型
JOB_TYPE = 'parttime'  # 兼职
```

### 3. 一键运行

```bash
python run_full.py
```

## 使用流程

### 方式一：一键运行（推荐）

```bash
python run_full.py
```

### 方式二：分步执行

```bash
# 1. 采集BOSS数据（包含公司主页信息）
python boss_spider.py

# 2. 采集企查查数据（可选）
python qichacha_spider.py

# 3. 整合数据
python data_merger.py
```

| 文件 | 说明 |
